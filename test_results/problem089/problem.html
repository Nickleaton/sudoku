Rule Explanation
FixedRatioPair A black dot between two cells means that one of the digits in those cells have a fixed ratio. The ratio is not necessarily an integer
Row Digits in each row must be unique
Column Digits in each column must be unique
FixedProductPair A red dot between two cells means that the digits in those cells have a fixed product
FixedSumPair Cells separated by an blue dot must have a sum the same sum
FixedDifferencePair A white dot between two cells means that the digits in those cells have a fixed difference.
Box Digits in each box must be unique
